# ExecPlans
Owner: Repo Maintainers
Last verified: 2026-02-12

This directory contains executable plans:
- `active/`: in-progress plans
- `completed/`: plans that shipped
- `tech-debt-tracker.md`: known debt, refactor targets, and recurring cleanup ideas

## Rules
- Any plan that changes direction must update its Decision Log.
- Any plan that discovers surprises must record evidence.
- Completed plans get moved to `completed/` with an Outcomes & Retrospective filled in.
